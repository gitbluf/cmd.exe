# PHASE 2: Plan-First Workflow

**Status:** Planning  
**Priority:** P0  
**Scope:** Implement `/dispath:plan`, `/dispath:synth`, `/dispath:apply` commands  
**Effort:** ~3 hours  
**Depends On:** Phase 1 (Agent Specialization)

## 📋 Overview

Implement a plan-first workflow where users can:
1. **`/dispath:plan <request>`** - Generate a plan via @blueprint (writes to `.ai/plan-<id>.md`)
2. **`/dispath:synth <request>`** - Execute plan from `.ai/plan-<id>.md` via @ghost
3. **`/dispath:apply <what to change>`** - Quick surgical edit via @ghost (no plan file needed)

This mirrors Neurogrid's workflow and eliminates generic "run an agent" invocations in favor of structured planning + execution.

### 2. Plan Registry (In-Memory + JSON)

Create `src/plans/registry.ts`:

```typescript
export interface PlanMetadata {
  id: string;                              // plan-abc123
  path: string;                            // ~/.pi/agent/dispath/.ai/plan-*.md
  title: string;                           // "Auth Refactor"
  request: string;                         // Original request
  status: "pending" | "executing" | "completed" | "failed";
  createdAt: ISO8601;
  completedAt?: ISO8601;
  executedBy?: "synth" | "apply";          // How it was executed
  error?: string;                          // If failed
  summary?: string;                        // First 200 chars of plan summary
}

export interface PlanRegistry {
  plans: PlanMetadata[];
  lastUpdated: ISO8601;
}

export function loadPlanRegistry(workspaceRoot: string): PlanRegistry;
export function savePlanRegistry(workspaceRoot: string, registry: PlanRegistry): void;
export function getPlan(workspaceRoot: string, id: string): PlanMetadata | null;
export function listPlans(workspaceRoot: string): PlanMetadata[];
export function createPlanId(): string;
export function createPlanFilename(title: string, id: string): string;
```

### 4. `/dispath:plan <request>` Handler

```typescript
async function handlePlanCommand(request: string, ctx: ExtensionContext) {
  // 1. Create plan metadata
  const planId = createPlanId();
  const filename = createPlanFilename("request", planId);
  
  // 2. Invoke @blueprint agent
  const blueprint = await loadAgent("blueprint");
  const planPrompt = `Generate a detailed plan for: ${request}
  
Format as a markdown plan file with:
- Summary
- Goals
- Files to change
- Implementation steps
- Risks
- Acceptance criteria

Be specific and actionable.`;

  const planContent = await blueprint.execute(planPrompt);
  
  // 3. Write plan file
  const planPath = path.join(workspaceRoot, ".ai", filename);
  fs.writeFileSync(planPath, planContent);
  
  // 4. Update registry
  updatePlanRegistry(planId, {
    path: planPath,
    status: "pending",
    request,
    createdAt: new Date().toISOString()
  });
  
  // 5. Show result in UI
  ctx.ui.notify(`📋 Plan created: ${planId}\nFile: ${filename}`);
}
```

### 5. `/dispath:synth <request>` Handler

```typescript
async function handleSynthCommand(request: string, ctx: ExtensionContext) {
  // 1. Find matching plan file
  const plans = listPlans(workspaceRoot);
  const planMeta = plans.find(p => p.title.includes(request) || p.id === request);
  
  if (!planMeta) {
    ctx.ui.notify(`❌ Plan not found: ${request}`);
    return;
  }
  
  // 2. Read plan file
  const planContent = fs.readFileSync(planMeta.path, "utf-8");
  
  // 3. Invoke @ghost agent with plan
  const ghost = await loadAgent("ghost");
  const synthPrompt = `Execute this plan:

${planContent}

Implement all the steps. Write code, edit files, and make all necessary changes.
Report what you've done.`;

  // 4. Execute with streaming output
  ctx.ui.notify(`🚀 Executing plan: ${planMeta.id}`);
  const result = await ghost.execute(synthPrompt, {
    onLog: (text) => ctx.ui.notify(text),
  });
  
  // 5. Update plan status
  updatePlanRegistry(planMeta.id, {
    status: "completed",
    completedAt: new Date().toISOString(),
    executedBy: "synth"
  });
  
  // 6. Show summary
  ctx.ui.notify(`✅ Plan executed: ${planMeta.id}\n${result}`);
}
```

### 6. `/dispath:apply <what to change>` Handler

```typescript
async function handleApplyCommand(instruction: string, ctx: ExtensionContext) {
  // 1. Invoke @ghost directly (no plan file)
  const ghost = await loadAgent("ghost");
  const applyPrompt = `Make this change: ${instruction}
  
Be precise and surgical. Only change what's needed.
Report what you changed.`;

  // 2. Execute with streaming
  ctx.ui.notify(`✏️ Applying change...`);
  const result = await ghost.execute(applyPrompt, {
    onLog: (text) => ctx.ui.notify(text),
  });
  
  // 3. Show result
  ctx.ui.notify(`✅ Change applied:\n${result}`);
}
```

### 7. `/dispath:plans` Handler

```typescript
async function handlePlansCommand(ctx: ExtensionContext) {
  const plans = listPlans(workspaceRoot);
  
  if (plans.length === 0) {
    ctx.ui.notify("No plans found. Use `/dispath:plan <request>` to create one.");
    return;
  }
  
  let output = "📋 Plans:\n\n";
  for (const plan of plans) {
    output += `- **${plan.title}** (${plan.id})
  Status: ${plan.status}
  Created: ${plan.createdAt}
  Summary: ${plan.summary || "N/A"}
\n`;
  }
  
  ctx.ui.notify(output);
}
```

### 8. `/dispath:clean` Handler

```typescript
async function handleCleanCommand(ctx: ExtensionContext) {
  // Remove all .md files in .ai/ directory
  const aiDir = path.join(workspaceRoot, ".ai");
  const files = fs.readdirSync(aiDir).filter(f => f.endsWith(".md"));
  
  for (const file of files) {
    fs.unlinkSync(path.join(aiDir, file));
  }
  
  // Clear registry
  savePlanRegistry(workspaceRoot, { plans: [], lastUpdated: new Date().toISOString() });
  
  ctx.ui.notify(`🗑️ Cleaned ${files.length} plan files`);
}
```
